# Sentential — 문장 기반 지식 관리 시스템

## 개요

지식 관리 시스템의 가장 큰 병목은 **지식을 추가하는 행위** 자체다. Sentential은 지식을 **완전하고 독립적인 문장(sentence)** 단위로 관리함으로써 이 병목을 해소한다. 하나의 문장은 그 자체로 의미가 완결되며, 다른 문장에 의존하지 않는다.

## 핵심 원칙

- **하나의 문장 = 하나의 지식**: 문장은 문맥 없이도 이해할 수 있어야 한다.
- **추가의 마찰을 최소화**: 입력창 하나, 엔터 한 번으로 지식이 저장된다.
- **검색이 곧 인터페이스**: 구글처럼 검색창 중심의 단순한 UI.

## 주요 기능

### 1. 지식 추가
- 검색창에 문장을 입력하고 저장한다.
- 저장 시 문장의 임베딩 벡터를 생성하여 함께 저장한다.

### 2. 지식 검색
- 임베딩 기반 시맨틱 검색으로 유사한 문장을 찾는다.
- 검색어와의 유사도 순으로 결과를 정렬한다.

### 3. 지식 수정 / 제거
- 검색 결과에서 문장을 선택하여 수정하거나 제거한다.
- 수정 시 임베딩 벡터를 재생성한다.

## UI / UX

- **구글 검색과 같은 단순한 형태**: 화면 중앙에 검색창 하나.
- **메인 컬러: 검은색** — 미니멀하고 집중력 있는 디자인.
- **모바일 우선 반응형**: 모바일에서도 유려하게 작동해야 한다.

## 기술 스택

| 계층 | 기술 |
|------|------|
| 프론트엔드 | Next.js |
| 백엔드 API | FastAPI |
| 벡터 DB | PostgreSQL + pgvector |
| 컨테이너 | Docker Compose |
| 도메인/DNS | Cloudflare |

## 인프라

- **Docker Compose**로 전체 시스템(프론트엔드, 백엔드, DB)을 자동 관리한다.
- 단일 `docker compose up` 명령으로 전체 환경이 구동된다.
- Cloudflare를 통해 도메인을 연결하고 HTTPS를 적용한다.
